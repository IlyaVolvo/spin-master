Regenerating screenshots (maintainers)

  1. Start the client so it can reach the API (for example npm run dev from the repo root).
  2. cd server && npm run seed:role-tutorials — upserts three demo logins in your database.
  3. cd server && ROLE_TUTORIAL_BASE_URL=http://localhost:3000 npm run capture:role-tutorials — writes PNGs under client/public/role-tutorials/assets/. Set PUPPETEER_EXECUTABLE_PATH if Chrome is not in the default location.

Demo logins share the password from server/scripts/seedRoleTutorialUsers.ts (constant ROLE_TUTORIAL_PASSWORD). Use only on non-production databases.

Static files are served at /role-tutorials/ when the Vite app or production build is running.
