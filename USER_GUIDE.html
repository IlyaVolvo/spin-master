Welcome to PingPong Tournament System

  3. The file should have the following columns (case-insensitive):

üë§ User Roles & Permissions

The system uses role-based access control. Different roles have different permissions:

Role Permissions
PLAYER
  • View all players and tournaments
  • Edit own profile (email, phone, address, picture, password)
  • View statistics and history
ORGANIZER
  • All Player permissions
  • Create tournaments
  • Add and edit match results
  • Complete tournaments
  • Delete tournaments and matches
  • Modify tournament names and participants
ADMIN
  • All Organizer permissions
  • Create, edit, and delete players
  • Edit all player fields (name, rating, roles, etc.)
  • Import and export players
  • Reset player passwords
  • Activate/deactivate players
  • View all members (not just players)
  • Cannot delete themselves
COACH
  • Same as Player permissions
  • May have additional coaching-specific features (if implemented)
Multiple Roles: Users can have multiple roles. For example, a user can be both a PLAYER and an ORGANIZER, giving them the combined permissions of both roles.
